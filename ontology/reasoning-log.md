# 온톨로지 추론 로그

이 파일은 온톨로지 추론 엔진이 생성한 추론 결과를 기록한다.
각 엔트리는 추론 규칙, 입력 트리플, 추론된 트리플, 신뢰도를 포함한다.

---

## 2026-04-07 추론 결과

### 추론 #1: event_chain (미사일 발사 연쇄)
- **입력:** (ent-007, MissileLaunch, 2026-01-04), (ent-008, MissileLaunch, 2026-01-27), (ent-009, MissileLaunch, 2026-03-14)
- **추론:** (ent-007 → ent-008 → ent-009) 시간순 이벤트 체인
- **신뢰도:** 0.85
- **상태:** 확정
- **근거:** 2026년 1~3월 사이 3차례 미사일 발사가 점진적 에스컬레이션 패턴을 보임

### 추론 #2: event_chain (무기 개발 연쇄)
- **입력:** (ent-009, MissileLaunch, 2026-03-14), (ent-010, FacilityActivity, 2026-03-28)
- **추론:** (ent-009 follows ent-010) 미사일 시험 → ICBM 엔진 시험 연쇄
- **신뢰도:** 0.80
- **상태:** 확정
- **근거:** KN-25 다연장로켓 시험 2주 후 ICBM 엔진 지상시험 실시, 무기 개발 프로그램 병행 추진

### 추론 #3: sanction_escalation (도발-제재 패턴)
- **입력:** (ent-009, type: MissileLaunch, 2026-03-14), (ent-014, type: SanctionAction, 2026-03)
- **추론:** (ent-014 triggeredEscalation ent-009)
- **신뢰도:** 0.75
- **상태:** 확정
- **근거:** 3월 14일 대규모 미사일 발사 직후 미국이 3월 내 추가 제재(IT 사기 관련)를 부과. 제재의 직접 명분은 사이버 활동이나 시기적으로 도발-제재 에스컬레이션 패턴에 부합

### 추론 #4: facility_weapon_link (시설-무기 연결)
- **입력:** (ent-003, developsAt, ent-002), (ent-010, relatedTo, ent-003)
- **추론:** (ent-002 linkedToLaunch ent-010)
- **신뢰도:** 0.70
- **상태:** 확정 (임계값)
- **근거:** 화성-20이 영변 핵단지와 연관되고, 엔진 시험이 화성-20과 연결 → 시설-시험 간접 연결
